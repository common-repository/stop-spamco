Luttez contre le Spam de commentaire.

== Description ==

Vous en avez marre de recevoir du spam dans vos commentaires ? Des visiteurs postent un commentaire de quelques mots seulement pour obtenir un lien ? Alors Stop SpamCo vous permettra de faire un premier pas dans la lutte contre le spamco.

Ce plugin vous laisse le choix entre retirer le champs 'Site web' après une periode que vous définissez. Ainsi les visiteurs ne pourront plus ajouter de lien dans les commentaires d'articles étant trop ancien.

Ou alors vous pouvez toujours autoriser les liens mais en passant le lien en nofollow pour tous les commentaires postés après la période que vous avez définit.

Version 0.2 :
Suppression d'une fonctionnalité empéchant les effets du plugin sur tous les articles postés avant l'installation du plugin Stop SpamCo


== Installation ==

1. Upload Stop SpamCo to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
